# UI 改进待办

## 聊天界面重构（用户反馈 2026-03-14）

### 问题
当前 ChatBox 的私聊交互方式不够直观——用户进入私聊模式后直接和某个 Agent 对话，缺少"选择聊天对象"的入口。

### 改进方向

#### 1. 增加联系人列表/聊天选择器
- 参考微信/WhatsApp 的交互模式：左侧是联系人列表，右侧是聊天窗口
- 联系人列表包括：
  - 各个 Agent（导购员、理货员、数据工程师等）
  - 聊天群（全员群聊，即当前的群聊模式）
- 每个联系人显示：头像/图标、名称、最近一条消息预览、未读标记
- 点击切换聊天对象

#### 2. 调度员也可以私聊
- 当前调度员只在群聊中作为路由角色出现
- 用户希望也能单独和调度员对话
- 场景：用户想直接问调度员"这个任务应该交给谁？"或"帮我协调几个Agent的工作"

### 实施优先级
- 这些是体验优化，不阻塞 Skills 系统开发
- 建议在 Skills 阶段一完成后再实施
